[球球大冒險]
遊戲介紹：
- 主要有兩種模式，”開始挑戰”為一般模式, 達成一定條件可進入星空模式來快速累績分數; ”飛高高吧”為歡樂模式，可無限跳躍且不扣血.
- 階梯共有8種顏色，白/灰/黃/紅/綠/紫/黑/淡藍+藍 /淡紫+紫 /彩虹色，各有不同功能，如下：
白色：扣血1格，會自動消失
灰色：分數加1分，短暫停留最多可回血1格，穿透時扣血1格
綠色：分數加2分，短暫停留最多可回血3格，穿透時不扣血
黃色：只出現在加分/星空模式，且穿透時回血1格, 其他功能和綠色一樣
紅色：連續扣血，直到離開為止
紫色：不扣血，會快速往上移動+自動消失
淡紫+紫色漸層: 扣血2格, 暈眩1.25秒, 穿透時扣血1格
淡藍+藍色漸層: 階梯會左右移動, combo數+3, 其他功能和灰色階梯一樣
黑色：隨機變換成黃色以外的8種顏色階梯
彩虹：踩到時，畫面中所有的階梯都變成綠色
- 切換方向鍵：有多種按鍵方法可以使用, 點擊可持續切換.
- 方向鍵“上”：掉落過程中，按“上”鍵可空中跳躍，扣血2格，最多2次；星空模式中或血量含5格以下使用，則不扣血.
- 方向鍵“下”：在階梯上，連續按“下”鍵2次，灰/綠/淡藍+藍色漸層/黃色階梯可穿透.
- 碰到上方針刺時, 扣血7格.
- 連續踩到灰或綠色階梯數次之後，球體會變成彩虹色，此狀態下，獲得分數為2倍.
- 在彩虹色球體狀態下，再連續踩到灰或綠色階梯數次之後，進入額外加分模式，可在一定時間內，可獲得大量分數.
- 開啟遊戲12分鐘後, 會跳出提示訊息要休息,密碼為動態=西元年月日時(24小時制).
